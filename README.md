## AI-Powered Interview Question Generator

An AI-based system that automatically generates customized technical and behavioral interview questions using Transformer models.

## Objective

To create role-specific interview question sets based on intern profiles and job descriptions.

## Model Used

GPT-2 (HuggingFace Transformers) with prompt engineering.

## Outcome

Generates intelligent, context-aware interview questions for AI-driven recruitment and internship screening.
